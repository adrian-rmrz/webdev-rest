About the Website

  • Framework:
    • Foundation: CSS framework used for all the styling
    • Vue: Javascript framework used for homepage and all the javascript
  • APIs:
    • Nominatim: Geolocating API; Used to find all the locations using either address or coordinates, will give both address and coordinates so it was used to convert from address to coordinates and vice versa
    • Leaflet: Map API; Used to generate the map and all the markers for locations
    • St. Paul Crime: Custom API; Used to grab crime data from St. Paul crime database
